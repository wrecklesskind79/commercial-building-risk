# Pricing Philosophy

Every [[Revenue Funnel]] layer is priced to be profitable standalone. No loss leaders. No "we'll make it up on the backend."

---

| Layer | Service | Model |
|---|---|---|
| [[Layer 1 — Assess]] | [[Commercial Inspections]] | Flat fee per property |
| [[Layer 2 — Plan]] | [[Preconstruction and Estimating]] | Flat fee or % of estimated construction cost |
| [[Layer 3 — Advise]] | [[Risk Consulting]] | Project fee (assessment) or retainer (ongoing advisory) |
| [[Layer 4 — Protect]] | [[NestShield Technology]] | One-time install + monthly recurring monitoring |

## Principles

- If a client only uses Layer 1, that's a profitable client
- Installation (Layer 4) covers costs and makes margin, but monthly recurring is the real business
- No hourly billing for inspections — flat fee, simple, predictable
- Every scope is written and signed before work starts. See [[Documentation Standards]]
