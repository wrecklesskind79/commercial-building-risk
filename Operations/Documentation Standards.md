# Documentation Standards

Every engagement, every [[Revenue Funnel|layer]], gets this. No exceptions.

---

1. Written scope/proposal before work starts
2. Signed authorization before work begins
3. Deliverable on time or early
4. Invoice on delivery
5. Follow-up on file

No handshake deals. No verbal-only scopes. Everything in writing. This protects you and builds the kind of practice that scales.
