# NestShield Technology

**Division:** [[NestShield]] (product division)
**Funnel Layer:** [[Layer 4 — Protect]]
**Revenue Model:** Install fee + monthly recurring monitoring

---

## What It Is

Commercial IoT platform for continuous building protection. Sensors detect water leaks and freeze conditions. Automatic shutoff valves stop the loss before it starts. 24/7 monitoring keeps someone watching.

## Capabilities

- 24/7 water leak monitoring
- Freeze monitoring
- Automatic water shutoff
- Commercial IoT sensor network
- Real-time alerting (email, text, app)
- Monitoring dashboard
- Monthly system health reports

## System Components

- Leak detection sensors (placed at high-risk locations)
- Freeze monitoring sensors (attics, exterior walls, unheated spaces)
- Automatic shutoff valves (sized for commercial supply lines)
- Connectivity (WiFi, cellular, or hardwired)
- Monitoring platform (cloud-based)

## Who Buys This

- Property owners with water loss history
- Property managers managing freeze-exposed buildings
- Insurance companies requiring/incentivizing monitoring
- Clients escalated from [[Risk Consulting]]
- Anyone who's had a pipe burst at 2 AM

## How It Feeds Back

NestShield monitoring data → reveals system degradation → new [[Commercial Inspections]]
Alerts show approaching failure → triggers [[Preconstruction and Estimating|estimate for replacement]]
Pattern across portfolio → triggers new [[Risk Consulting]] engagement

**The funnel is a loop.** See [[Revenue Funnel]].
