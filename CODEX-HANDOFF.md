# Codex Handoff

## What This Is

NestShield business — 4 pillars, 4 websites to build.

## Structure

```
sites/
  inspections/   — Commercial building inspections (CCPIA). Objective reports only.
  consulting/    — Repair consulting. 28 years of trade opinions.
  nestshield/    — IoT water risk management. The product. nestshield.net.
  estimating/    — Standalone freelance estimating. Overflow from other contractors.
```

## What's Built

- Content strategy and positioning for each pillar (in each README)
- Funnel logic: inspections → consulting → nestshield (connected). Estimating is standalone.
- Handoff language between pillars

## What's NOT Built

- Actual website code (each folder needs an index.html or framework)
- Domain setup (nestshield.net exists, others TBD)
- Visual design / branding
- Contact forms, scheduling, intake flows

## Rules

- Inspections report is OBJECTIVE — no opinions. That's the consulting upsell.
- Never pitch. Findings speak for themselves.
- Every pillar profitable standalone.
- Estimating has no required connection to the other 3.
- NestShield (IoT monitoring) is the endgame product. Everything else feeds it.
