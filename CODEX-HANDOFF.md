## The Core Idea

The business is a 4-layer revenue funnel:
1. Inspections (assess) → paid per inspection
2. Estimating (plan) → paid per estimate
3. Risk Consulting (advise) → paid per engagement
4. NestShield Technology (protect) → install fee + monthly recurring

Each layer is profitable standalone. Each layer's deliverable naturally surfaces the reason to go deeper. The funnel loops — NestShield monitoring data feeds back into new inspections.

## Rules for Extending This Vault

1. One concept per note. If a note covers two topics, split it.
2. Every new note must link to at least 2 existing notes.
3. Service/division notes link to their funnel layer, their career ladder section, and their pricing model.
4. New divisions follow the pattern in [[Future Divisions]] and [[Scalability]].
5. Career changes follow [[Career Framework]] — new positions map to existing classes.
6. Don't duplicate. Link.
